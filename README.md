# dsa-maze1

## Objective
Find the 3 separate paths to exit the maze.

## Instructions
1. `git clone https://github.com/samuraijane/dsa-maze1.git`
2. `cd dsa-maze1`
3. `node` to start a shell
4. At the prompt, enter `const run = require('./run.js');`
5. `run.findpath(0,0);`
